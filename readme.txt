This plugin will make it easy to embed a YouTube video in a post/page or sidebar widget.

== Description ==

This plugin is designed to help you easly display YouTube videos in your blog.  You have the option to show the video in a sidebar widget and in a post/page's content area.

== Installation ==
   * Upload post_video.php to plugin folder.
   * Activate from admin panel.
   * See Post Video under settings menu.
   * Set the default parameters. (You can overide default parameters by changing settings on post/page)
   * Create or edit a new post/page. At the end you will see a new section Post Video.
   * Put a video link there. Save the post. You will see the video in either homepage or in single post/page or sidebar widget.
   * The video link must look like: http://www.youtube.com/v/doZkxGXVt2E (on the YouTube page this code can be obtained from the embed box. For example look at the code below:
 <object width="425" height="344"><param name="movie" value="http://www.youtube.com/v/doZkxGXVt2E&hl=en&fs=1&"></param><param name="allowFullScreen" value="true"></param><param name="allowscriptaccess" value="always"></param><embed src="http://www.youtube.com/v/doZkxGXVt2E&hl=en&fs=1&" type="application/x-shockwave-flash" allowscriptaccess="always" allowfullscreen="true" width="425" height="344"></embed></object>
